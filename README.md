Wandrlust Twitter Harvester

* clone the repo
* create a new venv: `virtual venv`
* `source venv/bin/activate`
* `pip install -r requirements.txt`
* `python main.py`

add your credentials into `keys.txt` file in the format of

* 1: consumer_key
* 2: consumer_secret
* 3: access_token_key
* 4: access_token_secret
